### 1.**只要：**  

 > I don't mind where you go, as long as you're back before midnight.  
 > 只要你在午夜之前回来，你去哪儿都行。  （外研⑧ – 7）

 > One's life has value so long as one brings value to the life of others.  
 > 一个人只要给别人的生活带来价值，他的生命就是有价值的。  （2012 江苏改）
