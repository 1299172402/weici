### 1.*高义频：* **机会；时机 (chance)：**  

 > (**a**) **rare** / **golden** / **perfect opportunity**  
 > 难得的 / 绝佳的 / 理想的机会    

 > **take** / **seize** / **use** / **provide** / **miss an opportunity** (**to do sth**)  
 > 把握 / 抓住 / 利用 / 提供 / 错失 (做某事的) 机会
