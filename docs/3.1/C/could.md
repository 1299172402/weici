真题解析
---
1. I can’t find my purse. I______ it in the supermarket yesterday, but I’m not sure.   （2018 天津）  
A.should leave  
B.must have left  
C.might leave  
D.could have left  
2. It was so noisy that we _______ hear ourselves speak.   （2015 浙江）  
A.couldn’t  
B.shouldn’t  
C.mustn’t  
D.needn’t  
3. It ________ be the vocabulary that caused you the problem in the exercise because you know a lot of words.  （2013 安徽）  
A.may   
B.couldn’t  
C.should   
D.needn’t  
4. — Happy birthday!
— Thank you! It’s the best present I ______ for.   （2012 江苏）  
A.should have wished  
B.must have wished  
C.may have wished  
D.could have wished  

答案：
1. D  2. A  3. B  4. D
